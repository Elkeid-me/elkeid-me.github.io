每个人承担自己的风险！

2024-02-08
编程-Rust

Unwrap inner type when enum variant is known. From Stack overflow.

/// 每个人承担自己的风险！
#[macro_export]
macro_rules! risk {
    ($expression:expr, $pattern:pat => $extracted_expression:expr) => {
        match $expression {
            $pattern => $extracted_expression,
            _ => unreachable!(),
        }
    };
}

Example:


enum Type {
    I32(i32),
    I64(i64),
    F32(f32),
    F64(f64)
}

fn fun() -> Type {
    /// ... ...
}

let i = risk!(fun(), I32(i) => i);
