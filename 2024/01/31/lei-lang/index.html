雷 132

简介

雷语言是一种函数式编程语言：

f x y := x + y;
print "%d\n" (f 114 514); // => 628
0

if、which、while 是特殊的内置函数。 if 是条件分支：

x := 1;
if x = 1
(print "%d\n" 114)
(print "%d\n" 514); // => 114

if x = 2
(print "%d\n" 114) (
    if x = 1
    (print "%d\n" 514)
    _
); // => 514
0

鉴于 if 是函数，用在表达式是自然的：

z := 1;
x := if z = 1 3 2;
print "%d" x; // => 3

0

which 是多路条件选择：

x := input;
print_dec := print "%d\n";

if x = 1
(print_dec 1) (
    if x = 2
    (print_dec 2) (
        if x = 3
        (print_dec 3)
        (print_dec 4)
    )
);
// 上述嵌套 if 等同于 which:
which
(x = 1, print_dec 1)
(x = 2, print_dec 2)
(x = 3, print_dec 3)
(true, print_dec 4);

0

while 是循环，接受两个可调用对象为参数：

x :=~ 1;

while
(\ -> x < 5) (
    \ ->
    print "%d " x;
    x <- if x = 1 x + 2 x + 1;
    _
);
// =>
// 1 3 4

0

雷语言使用及早求值，但 if 和 which 使用惰性求值：

f x := (
    t := x + 1;
    print "%d\n" t;
    t
);

print "%d, %d\n" (f 114) (f 514);
// =>
// 115
// 515
// 115, 515

if true
(f 114)
(f 514); // => 115

0

雷语言支持柯里化：

print_hex := print "0x%x\n";
print_hex 1131796; // => 0x114514

0

雷语言规定：任何函数必须有参数。如果要实现无参数的函数，可以用关键字 _，例如：

p _ := print "114514\n";
p _; // => 114514

雷语言的表达式

let 表达式

let 表达式这个名字是历史原因，let 表达式里实际上没有 let。

变量 let 表达式
  • x := 1 表示定义一个名为 x 的不可变变量，值为 1；
  • x :=~ 1 表示定义一个名为 x 的可变变量，初值为 1；

变量 let 表达式的值为变量值本身。

函数 let 表达式
  • f x y := x + y 表示定义一个名为 f，接受参数 x 和 y 的函数。

函数 let 表达式的值为定义的函数本身。

元组 let 表达式

例如，(x, y, z) := (1, 2, 3) 表示对三元组 (1, 2, 3) 解包，x、y、z 的值分别为 1、2、3。

元组 let 表达式可以省略外层的括号，如上述代码也可写成 x, y, z := (1, 2, 3)。注意，雷语言中没有零元组和一元组。

元组 let 表达式可以用 _ 表示在解包的过程中忽略某个值，例如 x, _, z := (1, 2, 3) 表示对三元组 (1, 2, 3) 解包，x、z 的值分别为 1、3，而 2 则被忽略。

对于元组 let 表达式的值，以例子给出：

  • x, y, z := (1, 2, 3)，值是三元组 (1, 2, 3)
  • x, _, z := (1, 2, 3)，值是二元组 (1, 3)
  • x, _, _ := (1, 2, 3)，值是整型 1
  • _, _, _ := (1, 2, 3)，尽管没有人会这么写，但雷语言还是为这个 let 表达式赋予了值：_

赋值表达式

x <- expr，表示求值 expr，然后令可变变量 x 的值为 expr 的值。

类似地，也可以写为 expr -> x。

; 表达式

; 是一种二元运算符，定义如下：

  • expr_1 ; expr_2 表示按顺序求值 expr_1 和 expr_2，而整个 ; 表达式的值为 expr_2
  • ; 是右结合的，即 expr_1 ; expr_2 ; expr_3 解释为 expr_1 ; (expr_2 ; expr_3)。可以看到它按顺序求值 expr_1、expr_2 和 expr_3，整个表达式的值为 expr_3

例如：

f _ := (print "114 "; 114);
g x := 514 + x;
print ("%x"; "%d") (f _; g (f _)); // 114 114 628

0

函数调用表达式

expr_1 expr_2 expr_3 ... 表示先求值 expr_1、expr_2、expr_3、...（对于 if 和 which，则采用惰性求值），然后以 expr_2、expr_3、... 为参数调用 expr_1。

管道表达式

|> 是管道运算符，是一种二元运算符。

  • expr_1 |> expr_2 表示求值 expr_1 和 expr_2，然后 expr_2 以 expr_1 为参数调用。注意，由于柯里化的存在，expr_2 不一定为只接受一个参数的函数。
  • |> 是左结合的，即 expr_1 |> expr_2 |> expr_3 解释为 (expr_1 |> expr_2) |> expr_3。

Lambda 表达式

Lambda 表达式用于定义匿名的、一次性的函数。格式为 \ <参数列表> -> 函数体。

_ 对象

这是一种特殊的对象。类似 Ocaml 中的零元组。

示例

来看一个综合例子（print、range、map、filter、for_each 和 collect 是内置函数）：

f x := if x > 0 x + 1 x - 1;
p _ := print "114514 ";
range -10 10 |> map f |> filter (\ x -> p _; x < -5 \/ x > 0) |> for_each (print "%d\n");
// =>
// 114514 -11
// 114514 -10
// 114514 -9
// 114514 -8
// 114514 -7
// 114514 -6
// 114514 114514 114514 114514 114514 114514 114514 1
// 114514 2
// 114514 3
// 114514 4
// 114514 5
// 114514 6
// 114514 7
// 114514 8
// 114514 9
// 114514 10

// 等同于：
for_each (print "%d\n") (filter (\ x -> p _; x < -5 \/ x > 0) (map f (range -10 10)));

// 等同于：
10 |> (-10 |> range) |> (f |> map) |> ((\ x -> _ |> p; x < -5 \/ x > 0) |> filter) |> ("%d\n" |> print |> for_each);

快速排序：

// ~ 是列表连接运算符
q_sort a :=
    if (length a) = 0 []
    (a |> filter (\ x -> x < a[0]) |> collect |> q_sort)
     ~ (a |> filter (\ x -> x = a[0]) |> collect |> q_sort)
     ~ (a |> filter (\ x -> x > a[0]) |> collect |> q_sort);

[11, 4, 51, 4, 19, 191, 810] |> q_sort |> for_each (print "%d\n");
// =>
// 4
// 4
// 11
// 19
// 51
// 191
// 810

斐波那契数列：

fib x := (
    fib_helper x_0 x_1 n := (
        if n = 1
        (x_0, x_1, 0)
        (fib_helper x_1 x_0 + x_1 n - 1)
    );
    _, v, _ := fib_helper 0 1 x;
);

range 1 4 |> for_each (\ x -> print "%d " (fib x)); // => 1 1 2
